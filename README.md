# Projet-APP

## Installation des données

Avant de lancer le projet, vous devez télécharger les données et les placer directement dans le dossier `data/` à la racine du projet.

Structure attendue de data :
```
data/
├── Affiches
├── sujet1_f-42e0d11a
├── ...
```

## Installation des packages

- Créer un venv
- pip install -r requirements.txt
